# Wryft Landing Site

Marketing website for Wryft with legal pages.

## Setup

```bash
npm install
npm run dev
```

The site will run on http://localhost:3000

## Pages

- `/` - Homepage with features and download links
- `/terms` - Terms of Service
- `/privacy` - Privacy Policy
- `/guidelines` - Community Guidelines

## Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Deployment

You can deploy this to:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

Just point it to the `dist/` folder after building.
